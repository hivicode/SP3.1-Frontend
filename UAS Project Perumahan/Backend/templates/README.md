# Template Admin

Template Jinja untuk dashboard dan form admin:
- `login.html` - Form login admin
- `admin_base.html` - Layout dasar
- `admin_index.html` - Daftar properti
- `admin_form.html` - Form tambah/ubah properti + upload gambar
- `admin_customers.html` - Daftar booking dan ubah status

Semua template menerima data dari `app.py` dan menggunakan asset di `static/`.
