# Komponen Reusable

Fragmen HTML yang dapat di-include ulang di halaman frontend:
- `header.html` - Navigasi utama
- `header-minimal.html` - Header ringkas untuk halaman sekunder
- `footer.html` - Informasi kontak/perusahaan

Dapat digabung dengan mekanisme include sederhana (copy/paste atau templating tambahan) saat membangun halaman.
