# CSS Frontend

Kumpulan stylesheet untuk halaman frontend proyek perumahan:
- `index.css` - Styling landing page/hero
- `listing.css` - Layout dan filter untuk daftar properti
- `rent.css` - Styling halaman checkout/booking
- `contact.css` - Tata letak halaman kontak

Semua file menggunakan font, warna, dan spacing konsisten antar halaman. Impor CSS sesuai halaman yang sedang dibuka.
