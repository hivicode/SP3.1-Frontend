# Contoh 2 - Form Barang dengan Modal

Form input barang di dalam modal custom. Data disimpan pada array lokal dan dirender sebagai list.

## Berkas
- `index.html` - Tombol buka form, modal, dan list barang
- `style.css` - Styling modal dan layout
- `script.js` - Logika buka/tutup modal, submit form, dan render daftar

## Cara Pakai
1. Buka `index.html` di browser.
2. Klik "Tambah Barang" untuk membuka modal.
3. Isi kode/nama/harga, submit, dan lihat daftar terbarui.
