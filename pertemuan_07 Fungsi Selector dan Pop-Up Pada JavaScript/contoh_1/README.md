# Contoh 1 - Popup Overlay

Popup sederhana dengan overlay dan tombol tutup.

## Berkas
- `index.html` - Tombol buka popup dan container modal
- `style.css` - Styling overlay (`popup-tersembunyi/popup-tampil`) dan konten
- `script.js` - Event click untuk buka/tutup popup dan klik di luar konten

Buka `index.html` di browser lalu klik "Buka Pop-up" untuk mencoba.
