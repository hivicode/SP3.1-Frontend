# Template Flask

Template Bootstrap untuk contoh web dinamis pertemuan 12:

- `index.html` - Daftar produk dan tautan ke detail/cart
- `detail.html` - Detail produk (gambar, harga, stok, CTA cart)
- `cart.html` - Halaman keranjang sederhana

Semua template mengandalkan data yang dikirimkan dari `app.py` dan gambar di `/static/image/`.
