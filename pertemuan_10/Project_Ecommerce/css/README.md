# CSS - Project Ecommerce

`style.css` mengatur identitas visual halaman marketplace.

## Pokok Pengaturan
- Palet warna oranye, abu, dan putih via CSS custom properties
- Layout sticky header dengan brand, search box, dan tombol keranjang
- Panel filter dengan slider jQuery UI dan tombol kategori
- Grid kartu produk responsif (auto-fit minmax 240px)
- Styling dialog keranjang (`#cartDialog`) dan list item
- Breakpoint mobile (`max-width: 768px`) untuk header dan search box

## Catatan
Ubah variabel di `:root` untuk mengganti tema warna secara konsisten.
