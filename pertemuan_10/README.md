# Pertemuan 10 - jQuery UI Ecommerce

Contoh aplikasi ecommerce sederhana yang memanfaatkan jQuery UI untuk slider harga dan dialog keranjang. Fokus utama adalah filtering produk di sisi klien dan pengelolaan cart sederhana.

## Fitur
- Slider harga jQuery UI dengan label rentang yang dinamis
- Filter kategori berbasis data-attribute
- Dialog keranjang menggunakan `$(element).dialog()`
- Grid kartu produk responsif dengan badge, harga, dan metadata

## Struktur
- `Project_Ecommerce/` - Halaman utama marketplace
- `Project_Ecommerce/css/style.css` - Styling lengkap (header, grid, dialog)
- `Project_Ecommerce/js/script.js` - Logika filter dan cart

## Menjalankan
Buka `Project_Ecommerce/index.html` menggunakan HTTP server (mis. Live Server VS Code atau `python -m http.server`) agar resource eksternal termuat dengan benar.
