# Fungsi Dasar (Pertemuan 4)

Kumpulan contoh dasar fungsi JavaScript: parameter wajib/opsional, callback, rest parameter, dan penanganan overflow argument.

## Berkas
- `1_required_parameter.js` - Parameter wajib
- `2_optional_parameter` - Parameter opsional (tanpa ekstensi)
- `3_callback_function.js` - Callback sederhana
- `4_overflow_argument.js` - Menangani argumen berlebih
- `5_rest_parameter.js` - Rest parameter (`...args`)
- `pertemuan 4.js` - Ringkasan konsep fungsi

## Cara Pakai
Buka file `.js` dengan node atau browser console untuk melihat hasil console log.
