# Latihan Manajemen Barang

Latihan CRUD sederhana menggunakan fungsi JavaScript.

## Berkas
- `index.html` - Form input barang dan tabel daftar barang
- `fungsi.js` - Logika tambah, edit, hapus, dan reset data barang

## Menjalankan
Buka `index.html` di browser, isi form, dan gunakan tombol aksi untuk melihat perubahan daftar barang.
